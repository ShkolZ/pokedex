# Pokedex

This is a simple REPL that emulates pokemon game using Pokeapi. Anyone interested can try it out

## Features
- Location traversal
- Pokemon catching
- Pokemon stats
- Fight emulating(in progress)

## Installation
Install it from releases page
Should work on Linux, haven't tested others  

If you want to build binary yourself
1. Clone the project 
2. Open the project folder where the main.go is
3. Run 'go build'(if go doesnt get dependencies automatically run 'go mod download')

## Usage
1. Run pokedex
2. Type help to get more info
